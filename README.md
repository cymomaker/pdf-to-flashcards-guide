# PDF to Flashcards Guide

A practical guide for turning study PDFs into flashcards for active recall and spaced repetition.

## What This Guide Covers

- How to turn lecture notes, textbook chapters, and research papers into flashcards
- When to use Cloze cards vs Q&A cards
- How Anki APKG decks fit into a spaced repetition workflow
- How to review and edit generated cards before serious study

This guide is useful for students, exam candidates, researchers, and lifelong learners who want to convert dense PDF study material into reviewable flashcards.
